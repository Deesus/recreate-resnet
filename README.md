# ResNet implemented in TensorFlow/Keras

This notebook recreates the seminal [ResNet paper](https://arxiv.org/pdf/1512.03385.pdf) in TensorFlow. It uses the same model architecture, hyperparameters, and data augmentation methods as the original paper.

- [TensorFlow Addons](https://www.tensorflow.org/addons)
- [NumPy](https://numpy.org/doc/stable)
